# PARCIAL3_DTW-GRUPO8
Parcial 3 de Desarrollo Web
Escape Room JS — La Cámara de los Cinco Desafíos

La aplicación consiste en un escape room interactivo donde el usuario debe superar cinco niveles consecutivos para restaurar el acceso a un sistema crítico de una ciudad inteligente.

Integrantes
Bryan Ernesto Lemus Pineda — LP21019
Jeffrey Jabes Juarez Mangandi — JM21008
Luis Eduardo Molina Caceres — MC23015

La aplicación mantiene un flujo secuencial, cada desafío debe completarse correctamente para desbloquear el siguiente. Además, incluye una pantalla inicial de presentación, diseño visual por niveles, manejo de errores y exportación de resultados en formato JSON
